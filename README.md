# API École de Formation - NestJS

## Installation
1. `npm install`
2. Copier `.env.example` vers `.env` et renseigner les clés
3. Démarrer MongoDB
4. `npm run seed` (optionnel) ou `npm run start:dev`

## Endpoints
Base URL: `http://localhost:3000/api/v1`
Swagger UI: `http://localhost:3000/api/docs`
